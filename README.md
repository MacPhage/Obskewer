###What is Obskewer?
Obskewer is a light-weight Java application for encoding and decoding text using common ciphers.

Supported ciphers:
- ClassicShift (the form of the Caesar Cipher used with the initial release)
- The Caesar Cipher
- Morse code
- Letter-Numbers
- "Skip"
