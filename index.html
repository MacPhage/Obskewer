What is Obskewer?

Obskewer is a light-weight Java application for encoding and decoding text using common ciphers.

Supported Ciphers:

  • ClassicShift (the form of the Caesar Cipher used with the initial release of this software)
  • The Caesar Cipher
  • Morse code
  • Letter-Numbers (swapping alphabet letters with their position in the alphabet)
  • "Skip" (or a form of Scytale) (better described here)(coming soon)

To use, obtain the JAR file on the Releases tab and run it like any program (typically, you double-click on it). To quit the program, type "quit" when prompted.

This program works better with UTF-8, obviously.

Logo font is Paper Cuts 2. Used properly.
